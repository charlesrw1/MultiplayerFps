Test Modes:
	lateral steer mode:
		set AI's lateral target manually and watch it steer the course.
			`bool enable_course_boundary`: adds on layer to correct steering from going off course
		also can set AI "distance" from wheel to see it adjust with braking or accelerating
	paceline test
		manually set paceline mode of AI
	wheel test
		manually set what wheel AI follows, can force set the wheel also
